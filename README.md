Collection of cheats and patches for Gran Turismo 5 and 6. 

For RPCS3:
Download the .yml file and move the file over to the patches folder in your RPCS3 directory.
(GT6 1.05 Only) Use the .txt cheat codes as an alternative if the standard .yml memory patch doesn't work for any reason.

For PS3 (Requires CFW):
Download the .ncl tables and use them with NetCheat/Artemis.

Video showcase of the patch here: https://www.youtube.com/watch?v=ndvbrKmkF5Q
